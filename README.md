# İlk C# Projem: Sayı Toplama ve Ortalama

Bu proje, Bilgisayar Programcılığı 1. sınıf dersi kapsamında oluşturduğum ilk konsol uygulamasıdır.

## 🚀 Özellikler
* Kullanıcıdan iki farklı sayı alır.
* Girilen sayıları toplar ve ekrana yazdırır.
* Temel C# değişken yapılarını (int, string, double) ve tip dönüşümlerini içerir.

## 🛠️ Kullanılan Teknolojiler
* **Dil:** C#
* **Platform:** .NET Framework / Console App
* **IDE:** Visual Studio 2022

## 📝 Notlar
Hocanın tahtada gösterdiği "iki sayının ortalamasını alma" mantığına uygun olarak geliştirilmiştir.
